# Email Header Forensics

A browser extension for analyzing suspicious emails.

Extracts sender IP from headers and checks reputation on VirusTotal, IPinfo.io, and AbuseIPDB.

## Installation
1. Download or clone this repo
2. Go to `chrome://extensions`
3. Enable Developer Mode
4. Load unpacked extension folder

## Features
- IP extraction & reputation check
- Authentication (SPF/DKIM/DMARC)
- Risk scoring
- Suspicious script detection

**Note:** You need free API keys from the respective services.
